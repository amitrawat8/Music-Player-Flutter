# Music Player

## Supported devices
   - Android	Android 6.0 (API level 23) or higher.
   - iOS	    14 (all)

## Supported features
  - All Android	feature
   - iOS features

## Requirements to build the app.
   - Linux (x64)
  -  Windows (x64)
  - macOS (x64)

## Instructions to build and deploy the app
     - Clone
     - Install Flutter
     - Connect your Android device to your computer with a USB cable.
     - Enter cd [project] .
     - Run flutter pub get
     - Run flutter install .
